## Getting started

Using Zaap-sdk is as simple as adding this JavaScript code to your page:

```javascript
import { share } from 'zaap-sdk'

function App() {
  function handleShare() {
    await share({ message: 'Placeholder' })
  }

  return (
     ...
  )
}
```

## Installation

```sh
npm install zaap-sdk
# or
yarn add zaap-sdk
```
